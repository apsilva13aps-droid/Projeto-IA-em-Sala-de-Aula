# Projeto-IA-em-Sala-de-Aula
Repositório do Desafio relacionado ao Curso IA em Sala de Aula. Contém aplicação prática de uso do Chat GPT e Copilot, integrando os conhecimentos adquiridos ao decorrer das aulas. Aplica-se planejamento de aula, criação de materiais, diferenciação e internacionalização, observando-se ética e lei de proteção de dados, sempre com revisão humana.
